GOOGLE SEARCH CONSOLE VERIFICATION (HTML FILE METHOD)
====================================================

1. Go to https://search.google.com/search-console
2. Add property → URL prefix → https://postpilot.company
3. Choose "HTML file upload" as verification method
4. Download the verification file Google provides (e.g. google1234567890abcdef.html)
5. Put that file in this folder (public/)
6. Deploy to Vercel
7. Back in Search Console, click "Verify"

The file will be served at: https://postpilot.company/[filename].html
